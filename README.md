# Initech AI Course

Responsive landing page for an AI fundamentals course. Built with React, Vite, and Tailwind CSS.

## Requirements
- Node.js 20+
- npm

## Setup
```bash
npm install
```

## Development
```bash
npm run dev
```
Open http://localhost:5173.

## Build and Preview
```bash
npm run build
npm run preview
```

## Deployment
- GitHub Actions deploys to GitHub Pages on pushes to `main`.
- In GitHub, set Pages source to "GitHub Actions".
- Ensure `base` in `vite.config.js` matches the repository name.

## Content Updates
- Edit copy and layout in `src/components/LandingPage.jsx`.
- Placeholder values are listed in `PLACEHOLDERS.md`.
